基于hexo+GitHub的博客搭建

发表于 2024-01-04 分类于 github使用 本文字数： 1.3k

介绍博客的搭建以及部分美化

过程

安装git和node.js

[git官网] (https://git-scm.com/downloads)下载git并安装，在git中绑定GitHub的用户名和邮箱
[node官网] (https://nodejs.org/en/download/)下载并安装，配置环境变量

安装hexo

安装命令行：
npm install - g hexo-cli
初始化命令行：
hexo init myBlog
cd myBlog
npm install

加载主题并配置

在Hexo官网搜索自己喜欢的主题样式，将站点文件夹下的_config.yml中的theme修改为所选主题名称
hexo s 本地4000浏览

部署github

创建同名仓库（如pfmirr.github.io其中pfmirr为username）
配置ssh key如前三行命令,首次使用还须要确认并添加主机到本机SSH可信列表,如第四行命令
git config –global user.name “用户名”
git config –global user.email “邮箱地址”
ssh-keygen -t rsa -C ‘上面的邮箱’
ssh -T git@github.com
登陆 Github 上添加刚刚生成的SSH key
链接本地和github,如下：
deploy:
type: git
repo: git@github.com:pfmirr/pfmirr.github.io.git
branch: master
安装部署插件 hexo-deployer-git，如下：
npm install hexo-deployer-git –save
生成并部署，采用pfmirr.github.io查看
hexo g
hexo d

美化

隐藏底部强力驱动

vscode全局搜索并注释：ctrl+F 主题配置文件powered和enable改为false

站内搜索文章

安装插件hexo-generator-searchdb
站点配置文件添加如下：
#站内搜索
search:
path: search.xml
field: post
format: html
limit: 10000

主题配置文件修改如下：
local_search:
enable: true

背景的设置

主题背景的source/css/main.styl,代码展示如下：
//自定义背景样式
body {
background: url(/images/backs.jpg);
background-repeat: no-repeat;
background-attachment: fixed;
background-position: 50% 50%;
color: var(–text-color);
font-family: ‘Lato’, “PingFang SC”, “Microsoft YaHei”, sans-serif;
font-size: 1em;
line-height: 2;
}
